# Power BI 助手 — 使用说明

AI 辅助编写 / 校验 Power BI 的 DAX 度量值。生成的每条度量值都会连到你**正在打开的** Power BI Desktop 实跑验证，可一键写回模型。

## 一、运行前提
1. **Windows 电脑**（不支持 Mac）。
2. **已安装 Power BI Desktop**，并且**已打开要分析的报表**（本工具靠它读取模型、实跑验证）。
3. 一个**大模型 API Key**（如通义千问 / DashScope），**每人用自己的**。

## 二、启动
1. **解压到一个短路径目录**，例如直接放在 `C:\PowerBI助手`。
   > ⚠️ 重要：**不要**解压到很深 / 很长的文件夹（如桌面里再套好几层、或 OneDrive 深层目录）。Windows 有 260 字符路径上限，路径太长会导致部分文件释放失败、应用报错。放在 `C:\` 下最稳。
2. 双击 **`packaging\启动.bat`** 启动。会先出现一个命令行窗口（正常），稍后自动弹出**应用窗口**。
   - 第一次启动若在准备环境，请联网等待几分钟（仅首次）。
   - **想要"无黑窗"的干净体验**：请改用**安装包版**（`PowerBI助手_安装.exe`），装好后从开始菜单的「Power BI 助手」启动，没有命令行窗口。
   - 注意：**请勿直接双击 `启动.vbs`**——很多电脑把 `.vbs` 默认用记事本打开（只会弹出代码、不会运行）。要无黑窗请用安装包，或右键 `启动.vbs` →「打开方式 →  Microsoft Windows Based Script Host」。
3. **关闭应用窗口（或那个命令行窗口）= 关闭应用**，服务会自动停止。

## 三、首次配置
1. 点左上角的 **⚙ 设置**，选择大模型供应商、填入**你自己的 API Key**，保存。
   - Key 仅保存在你本机（`C:\Users\你的用户名\.powerbi_ai_assistant\`），不会上传。
2. 左侧「数据源」红灯 → 点 **连接 Power BI Desktop**，变绿灯即连上当前打开的报表。

## 四、使用
- **AI 对话**：用自然语言描述要的度量值，回车发送；左侧点字段名可插入引用。
- **校准式生成**：你给出某个切片下"已知正确的值"，AI 反复验证逼近正确口径。
- 开「✍️ 写入 Power BI」开关后，验证通过的度量值/计算表可一键写回。**写回后在 Power BI Desktop 里按 Ctrl+S 才会真正保存。**

## 五、常见问题
- **浏览器打开是错误页/连不上**：稍等几秒刷新，服务还在启动。
- **左侧连不上 Power BI**：确认 Desktop 已打开报表；切换报表后点一下「连接」重连（端口每次重开会变，工具会自动找）。
- **第一次写回度量值较慢/报错**：首次写回需联网下载一个 ~30MB 的组件（仅一次）；公司网络受限时可能需要稍候或重试。
- **AI 回复慢**：把「🧠 深度思考」开关关掉（默认就是关的），简单问题会快很多。
